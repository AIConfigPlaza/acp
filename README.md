# ACP (AI-Config-Plaza)

## 项目概述

**ACP (AI-Config-Plaza)** 是一个专注于简化AI编程工具配置的平台，旨在解决开发者在使用各类AI编程工具时面临的配置碎片化、重复劳动多、跨工具切换成本高的问题。通过统一的配置管理和社区共享机制，让AI工具配置更高效、更简单。

## 核心功能

### 1. 多类型配置统一管理
- **支持多种配置格式**：涵盖MCP配置、Agent配置（如`Agents.md`）、自定义指令（Prompt）等
- **灵活的配置属性**：每种配置可设置名称、描述、标签、公开/私有权限，支持点赞、下载统计

### 2. 一站式方案市场
- **组合配置方案**：将多个相关配置（如一个Agent配置+多个MCP配置+自定义指令）打包为完整方案
- **工具适配**：方案与主流AI工具（Claude Code、Copilot、Codex、Cursor等）绑定，确保兼容性

### 3. 一键式CLI工具

### 4. 社区共享与协作
- **配置分享**：支持将个人配置/方案设为公开，供社区使用
- **互动反馈**：通过点赞反馈优质配置，帮助其他开发者筛选
- **个人中心**：管理自己发布的配置、收藏的资源，跟踪下载和使用数据


### 5. 跨工具兼容
- 无缝支持Claude Code、GitHub Copilot、OpenAI Codex、Cursor等主流AI编程工具
- 统一的配置导入/导出格式，降低工具切换时的配置迁移成本
